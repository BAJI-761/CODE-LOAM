# CodeLoom 4-Minute Demo Script

## 0:00-0:30 — Introduction & Value Prop
- **Screen**: Landing Page (unauthenticated)
- **Talking Track**: "Welcome to CodeLoom. We built this for the Sqrock IT Solutions internship to solve the fragmentation in online learning. Instead of using one platform for videos, another for coding, and another for AI help, CodeLoom unifies them into a single, seamless experience."

## 0:30-1:30 — Student Experience: Learning & Coding
- **Screen**: Student Dashboard -> Resume a Course -> Lesson -> Challenge Workspace
- **Action**: 
  1. Click into a course and show the video player/markdown lesson.
  2. Navigate to a coding challenge.
  3. Write a deliberate syntax error in the Monaco Editor and hit "Run".
- **Talking Track**: "Here's the student view. We can consume rich multimedia lessons. But the real magic is the interactive coding workspace. Let's try this challenge. I'll make a deliberate mistake and run the code. As expected, it fails the test cases."

## 1:30-2:30 — AI Integration (The "Aha!" Moment)
- **Screen**: Challenge Workspace -> Click "AI Hint" -> Open Global Chat
- **Action**: 
  1. Click the "AI Hint" button.
  2. Show the Global AI Chat sliding in.
  3. Show the AI giving a contextual hint based on the code in the editor, without giving the direct answer.
  4. Fix the code, hit "Submit", and get the success toast.
- **Talking Track**: "Instead of getting stuck, the student can click 'AI Hint'. Our Gemini integration is context-aware—it reads the code in the editor and provides a pedagogical hint without just giving away the answer. I'll fix the code based on the hint, submit it, and boom, we pass all test cases and earn points!"

## 2:30-3:15 — Instructor Experience
- **Screen**: Instructor Dashboard -> Create Course/Quiz
- **Action**: 
  1. Log out, log in as Instructor.
  2. Show the analytics dashboard (Total students, average completion).
  3. Go to course creation, show the "Generate Quiz with AI" button.
- **Talking Track**: "Switching to the Instructor role, we have a powerful dashboard to track student performance. Instructors can also leverage AI to generate quizzes instantly based on a topic, saving hours of manual content creation."

## 3:15-4:00 — Gamification & Conclusion
- **Screen**: Student Certificates Page -> Download PDF
- **Action**: 
  1. Switch back to Student (or have another tab ready).
  2. Go to Certificates page.
  3. Download the high-res PDF certificate.
- **Talking Track**: "Finally, when a student reaches 100% completion, CodeLoom auto-generates a verifiable, high-resolution PDF certificate. This drives engagement and gives students a tangible reward for their hard work. That's CodeLoom—a complete, AI-powered LMS."
